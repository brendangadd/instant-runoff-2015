What would happen if Canada used instant runoff vote for its electoral system?

On October 19, Canada elected a new government using a first-past-the-post system (i.e. vote for one candidate, winner takes all).

These days, there is a lot of talk about electoral reform. One possible alternative is instant runoff voting. So what would happen if Canada's forty-second election used instant runoff voting instead of first-past-the-post? This simulation tries to answer that question.
